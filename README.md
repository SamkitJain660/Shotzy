# GnomeGoogleLens
image search on google lens directly from the built in gnome screenshot tool

![Gnome Google Lens Showcase](./gnome_google_lens.gif)

### Installation
* **Direct Download:** Install it through the official portal:
    [Pending Approval...](https://extensions.gnome.org)

### Manual Installation
1. Create the local extensions directory if it does not already exist:
   ```bash
   mkdir -p ~/.local/share/gnome-shell/extensions
   ```
2. Clone this project into the GNOME Shell extensions directory:
   ```bash
   git clone https://github.com/SamkitJain660/GnomeGoogleLens.git ~/.local/share/gnome-shell/extensions/google-lens-search@SamkitJain660.github.io
   ```
3. Log out and log back in.

4. Enable the extension:
   ```bash
   gnome-extensions enable google-lens-search@SamkitJain660.github.io
   ```
